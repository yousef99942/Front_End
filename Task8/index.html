What is the difference between checkbox and radio?

In the beginning, bot of them are for choose form choices. But we can say that there is two difference between them:

1- In checkbox, we can choose one or more choices.
In radio, we can choose just one choice.

2- In checkbox, it uses a square shape for the choices.
In radio, it uses a circle shape for the choices.
